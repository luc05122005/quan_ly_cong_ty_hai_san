## 📌 Introduction

**Seafood Company Website** is a full-stack web application designed to support seafood businesses in:
- Showcasing fresh seafood products
- Managing customers and orders
- Providing a smooth and user-friendly online shopping experience

This project is built as an **academic project / portfolio website**, focusing on **clean architecture, database integration, and real-world business logic**.

---

## ✨ Features

### 👤 Customer
- Browse seafood products by category (fish, shrimp, crab, squid, etc.)
- View product details with images and prices
- Add products to shopping cart
- Update quantity & checkout
- User registration & login
- View order history

### 🛠️ Admin
- Secure admin login
- Manage products (add / edit / delete)
- Manage categories
- View customer list
- View and manage invoices
- Sales statistics & revenue charts
